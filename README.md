Este projeto é o que esta sendo desenvolvido no mini curso que esta disponível no youtube! :star2:

# Pré requisitos
* Java 8 ou 9
* Maven 3 ou 5
* JUnit 5

# Para executar o projeto
`mvn clean install`

## Para deploy na aws basta ter o gitbash e seguir o tutorial do último vídeo
